# Детекция автомобильных номерных знаков: сравнительный анализ архитектур компьютерного зрения

## Описание проекта

Данный проект посвящён исследованию и сравнительному анализу современных архитектур нейронных сетей для задачи детекции автомобильных номерных знаков.

В рамках работы были обучены, протестированы и сравнены пять моделей:

* YOLOv8n
* YOLOv8s
* YOLOv8m
* RT-DETR
* Faster R-CNN

Целью исследования являлся выбор архитектуры, обеспечивающей наилучшее соотношение точности детекции, скорости работы и вычислительных затрат.

По результатам сравнительного анализа был разработан сервис обнаружения автомобильных номерных знаков, включающий REST API, веб-интерфейс и систему хранения истории запросов.

---

## Основные возможности

* обучение и тестирование современных моделей детекции объектов;
* сравнительный анализ архитектур компьютерного зрения;
* расчёт и визуализация метрик качества;
* REST API для обработки изображений;
* веб-интерфейс для взаимодействия с моделью;
* хранение истории запросов и статистики;
* контейнеризация с использованием Docker.

## Результаты сравнительного анализа

### Качество моделей

| Модель       | mAP@0.5 | mAP@0.5:0.95 | Precision | Recall |    F1 |
| ------------ | ------: | -----------: | --------: | -----: | ----: |
| YOLOv8n      |   0.992 |        0.818 |     0.983 |  0.974 | 0.978 |
| YOLOv8s      |   0.991 |        0.828 |     0.984 |  0.975 | 0.980 |
| YOLOv8m      |   0.991 |        0.830 |     0.980 |  0.981 | 0.981 |
| RT-DETR      |   0.987 |        0.832 |     0.968 |  0.969 | 0.968 |
| Faster R-CNN |   0.957 |        0.875 |     0.957 |  0.925 | 0.941 |

### Производительность моделей

| Модель       | Время инференса (мс) | Размер модели (МБ) |
| ------------ | -------------------: | -----------------: |
| YOLOv8n      |               299.19 |               5.96 |
| YOLOv8s      |               778.78 |              21.47 |
| YOLOv8m      |              1651.78 |              49.62 |
| RT-DETR      |              5161.43 |              63.16 |
| Faster R-CNN |             11992.49 |             471.47 |

### Основные выводы

* YOLOv8n продемонстрировала наилучшее соотношение точности и скорости работы.
* Более крупные модели обеспечивают незначительный прирост качества при существенном увеличении вычислительных затрат.
* Faster R-CNN показала высокую точность локализации, однако обладает наибольшими требованиями к вычислительным ресурсам.
* RT-DETR обеспечивает конкурентоспособное качество детекции, но уступает семейству YOLO по скорости работы.
* Для практического применения была выбрана модель YOLOv8n.
